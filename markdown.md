                    3.2 เมื่อตรวจสอบเสร็จแล้ว ผมจะเริ่ม แต่ผมกลับเจอปัญหาคือ ถ้าเราสร้าง Array แล้วเก็บตัว Field มันก็จะตายตัว เหมือนเกม Rockman ที่มันมี State อยู่แล้ว ผมอยากได้แบบ Randdom แต่พอทำ Random มันก็จะเปิดบักเกี่ยวกับเส้นทางที่ปิด หรือช่องเต็มทำให้ไม่สามารถสร้างเส้นชัยได้ ดังนั้นผมจึงต้องใช้ตัวช่วยถาม AI ว่ามันมีวิธีไหม (ผมเคยลองเขียนบอทจับสีทางเดิน แล้วมันเกิดปัญหาแบบนี้) 
                    3.3 ผมไปซื้อกาแฟมาดื่ม ผมก็คิดว่าถ้าทำแบบดีๆ ไปเลยก็กลัวงานจะไม่เสร็จ ผมคิดได้ว่าถ้าอย่างงั้นทำแบบนี ผมจะทำเป็น2 แบบ

                                    • แบบ State ตัวอย่างให้ลองเล่นซึ่ง 2 State ให้เลือก คือ 3*3 และ 4*4 

                                    • แบบ Random สายประนีประนอม: การันตีทางเดิน (Guaranteed Path)

                            กรณี > ผู้เลือกเล่นเกมเลย
                            จะเล่นเกมจริงๆ เลย โดยการเล่นเกมจริงจะเป็นการ Random แบบสายประนีประนอม: การันตีทางเดิน (Guaranteed Path)
